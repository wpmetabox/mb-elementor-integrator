## Description

[Meta Box](https://metabox.io) lets you add **custom fields** to WordPress - for posts, pages, custom post types, terms, users, and settings pages. [Elementor](https://elementor.com) is one of the most popular page builders, and its **dynamic tags** feature lets you pull dynamic content into your designs.

This plugin connects the two. It adds a **"Meta Box Field"** dynamic tag to Elementor, so you can display any Meta Box custom field directly in your Elementor-built pages - without writing code.

### Why you need it

Elementor's dynamic tags support WordPress core content like the post title or featured image, but they do **not** understand Meta Box custom fields out of the box. To show a custom field, you normally have to know and type its exact **field ID**, which is easy to mistype and hard to remember.

With this plugin, you simply **pick the field from a dropdown list**. No need to remember field names or IDs. It just works - for posts, archive pages, and site-wide templates.

### What it does

- Adds a **Meta Box Field** dynamic tag to Elementor's dynamic tags picker.
- Works with **all field types** - text, images, files, WYSIWYG, taxonomies, relationships, and more.
- Supports **groups**, **cloneable groups**, and **sub-fields**.
- Displays custom fields from **posts**, **terms**, and **settings pages**.
- Outputs formatted values automatically, including images, links, and HTML.

## Installation

You need to install [Meta Box](https://metabox.io) plugin first

- Go to **Plugins | Add New** and search for "Meta Box"
- Click **Install Now** to install the plugin
- After installing, click **Activate Plugin** to activate it

Install **MB Elementor Integration** extension

- Go to **Plugins | Add New** and search for "MB Elementor Integration"
- Click **Install Now** to install the plugin
- After installing, click **Activate Plugin** to activate it
